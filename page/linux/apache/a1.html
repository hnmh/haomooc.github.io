Ubuntu安装Apache +Php


Ubuntu安装Apache


1 安装apache，在命令行终端中输入一下命令：【sudo apt-get install apache2】


2 安装完成后，重启apache服务，在命令行终端中输入一下命令 【sudo /etc/init.d/apache2 restart】


Ubuntu安装Php(我是安装的最新版)


3在命令行终端中输入一下命令 【sudo apt-get install php y 】


4 让Apache支持php,在命令行终端中输入一下命令 【sudo apt-get install libapache2-mod-php】


5 安装php5-gd模块，在命令行终端中输入一下命令 【sudo apt-get install php-gd】


默认web目录 【/var/www/html】


完成
