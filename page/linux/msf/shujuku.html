msf链接postgresql数据库


    操作环境：Unix/Linux系列


    安装postgresql数据库：apt-get install postgresql


    启动服务：service postgresql start


    修改postgresql密码为Sql@2017：{{{


        sudo -u postgres psql postgres


        root@kali:~# sudo -u postgres psql postgres


        postgres=# alter user postgres with password 'Sql@2017';


        修改成功会看到：ALTER ROLE


    }}}


    启动msf：msfconsole


        msf > db_status


        [*] postgresql connected to postgres


        msf > db_connect


        {{{


        [*] Usage:


        [*] Examples:


        [*] db_connect user@metasploit3


        [*] db_connect user:pass@192.168.0.2/metasploit3


        [*] db_connect user:pass@192.168.0.2:1500/metasploit3


        }}}


        msf > db_connect postgres:Sql@2017@127.0.0.1:5432/postgres


    总结：整个过程就几句：


    apt-get install postgresql && service postgresql start


    sudo -u postgres psql postgres


    alter user postgres with password 'Sql@2017';


    msfconsole


    db_connect postgres:Sql@2017@127.0.0.1:5432/postgres
