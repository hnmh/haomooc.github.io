ubuntu开启指定端口


首先我们以腾讯云ubuntu服务器为例子(我的默认规则是只对外开放22的ssh端口)：


1】首先我们增加一条过滤规则：这条规则是对外允许25800端口通信：
sudo iptables -A INPUT -p tcp -d 0/0 -s 0/0 --dport 25800 -j ACCEPT


2】然后设置防火墙(我的服务器默认是关闭状态的)：
2.0）查看防火墙状态：sudo ufw status
2.1）开启防火墙：sudo ufw enable
2.2）允许25800通过：sudo ufw allow 25800
2.3）重启防火墙：sudo ufw reload
