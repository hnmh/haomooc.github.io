Nginx安装


  Ubuntu下安装：sudo apt-get install nginx  若你是root用户，可以直接使用apt-get install nginx 不用加sudo命令


  Ubuntu下启动：sudo /etc/init.d/nginx start 其他信息：sudo /etc/init.d/nginx start /？


  这个时候：如果访问不了你的nginx服务，查看监听端口netstat -ano ,那么也请检查你的80端口是不是被屏蔽？


  附加信息：


  Nginx服务地址：/etc/init.d/nginx　


  配置地址：/etc/nginx/　　如：/etc/nginx/nginx.conf


  web默认目录：/var/www/html　　如：/var/www/html/index.nginx-debian.html;


  现在可以运行静态网页，请修改/var/www/html下的文件。
